1 /**
2  * @file adc.h
3  * @ingroup io
4  *
5  * Library for interfacing with AD converter.
6  *
7  * The AD converter library supports a moving average filter for each sampled
8  * channel. Configuration of the AD converter library is included in
9  * src/cfg/io/adc/adc_cfg.h where size of the moving average filter
10  * (#ADC_FILTER_SIZE), number of channels to be sampled (#ADC_CHANNEL_COUNT) and
11  * specification of channels (#ADC_CHANNEL_INIT) is given.
12  *
13  * After calling ADC_init() the ADC is setup to sample all given channels
14  * consecutively using the ADC complete interrupt. The ADC clock is set to a
15  * frequency of 62.5kHz which is the lowest possible frequency resulting in the
16  * least noise of sampled values. The first conversion takes 25 ADC clock cycles
17  * (400us). All following conversions take 13 ADC clock cycles each (208us).
18  *
19  * Starting a conversion is triggered through timer 5 compare match ISR
20  * <code>TIMER5_COMPA_vect</code> (see lib/tools/timeTask/timeTask_isr.S) every
21  * two milliseconds. As soon as a conversion is finished (after 208us or 400us),
22  * the ADC complete ISR <code>ADC_vect</code> (see lib/io/adc/adc_isr.S) is
23  * executed which reads the converted value and stores it in a data structure
24  * which contains the sampled values for each channel and their average value.
25  * The ISR then sets up the ADC for the conversion of the next channel which
26  * will be triggered by run of <code>TIMER5_COMPA_vect</code> after the next.
27  * So every two milliseconds, a single channel is sampled. This means that the
28  * frequency of sampling each channel depends on the number of overall channels
29  * to be sampled. Thus, each channel is sampled every #ADC_CHANNEL_COUNT * 2ms.
30  *
31  * The latest sampled raw value for a given channel can be requested with
32  * ADC_getLastValue(). Note that the age of this value is in the
33  * worst case about #ADC_CHANNEL_COUNT * 2ms.
34  *
35  * The filtered sample can be read with ADC_getFilteredValue().
36  * This is a result of applying a moving average filter to the latest
37  * #ADC_FILTER_SIZE samples. Note that the first meaningful value can be
38  * obtained after the ADC has sampled a channel at least #ADC_FILTER_SIZE times.
39  * Thus, measured from the time when ADC_init() was called and interrupts were
40  * globally enabled, the first readout of the filtered value should occur no
41  * earlier than (#ADC_FILTER_SIZE * #ADC_CHANNEL_COUNT * 2ms) + 2ms.
42  *
43  * Note that filtering introduces a time delay. The group delay of the moving
44  * average filter is exactly (#ADC_FILTER_SIZE - 1) / 2 samples or
45  * (#ADC_FILTER_SIZE - 1) / 2 * #ADC_CHANNEL_COUNT * 2ms.
46  *
47  * The JTAG interface is enabled by default up on a power cycle. When enabled,
48  * the pins PF4 to PF7 (TCK, TMS, TDO, TDI) which also have the alternate
49  * functions ADC4 to ADC7 can only be used for the JTAG interface. In this
50  * situation all these pins are pulled high by the hardware and cannot be used
51  * by the ADC.
52  *
53  * In order to allow their usage by the ADC, the JTAG can be switched off by
54  * calling ADC_init() with the argument true. The JTD bit of the register MCUCR
55  * is then used to turn off the JTAG interface.
56  */

69 /**
70  * Initialize the AD converter library.
71  *
72  * Global interrupts must be enabled manually after calling this function.
73  *
74  * See src/cfg/io/adc/adc_cfg.h for configuration of the AD converter library.
75  *
76  * The parameter disableJTAG allows disabling the JTAG interface, so that pins
77  * PF4 to PF7 (ADC4 to ADC7) can be used for AD conversions.
78  *
79  * Note that the JTAG interface is enabled by default up on a power cycle. When
80  * enabled, the pins PF4 to PF7 (TCK, TMS, TDO, TDI) which also have the
81  * alternate functions ADC4 to ADC7 can only be used for the JTAG interface.
82  * In this situation all these pins are pulled high by the hardware.
83  *
84  * In order to allow their usage by the ADC, the JTAG needs to be switched off
85  * using the JTD bit of the register MCUCR. A timed sequence is necessary in
86  * which the new value of MCUCR must be written twice within four clock cycles.
87  * Thus, interrupts will be disabled before and re-enabled afterwards if they
88  * were enabled before.
89  *
90  * @param disableJTAG true to disable JTAG and allow pins ADC4 to ADC7 to be
91  * used by the ADC, false to keep JTAG enabled
92  */
93 void ADC_init(const bool disableJTAG);
