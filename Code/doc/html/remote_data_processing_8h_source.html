17 /**
18  * Send a command on communication channel #CH_OUT_RDP to a remote data
19  * processor (for example MATLAB, Octave, Scilab) through HWPCS. The commands
20  * are added to a list (a script to be executed by a configured data processor)
21  * within HWPCS as long as the parameter execute is false. In order to add
22  * the last command and execute the whole script, set the parameter execute
23  * to true.
24  *
25  * @param execute false to just add the command to the list,
26  * true to add the command and execute the whole script
27  * @param command a string containing the command, for format please see
28  * <a href="https://www.nongnu.org/avr-libc/user-manual/group__avr__stdio.html" target="_blank">vfprintf</a>
29  * from AVR Libc.
30  * The resulting string will be truncated after 512 characters
31  * excluding null termination.
32  *
33  */
34 void remoteDataProcessing_command(const bool execute, const char* command, ...);
